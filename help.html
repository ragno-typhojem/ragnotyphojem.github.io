Installation Guide

Welcome to KodKit! This guide will walk you through the process of installing KodKit on your Windows computer.

System Requirements

Before installing KodKit, make sure your computer meets the following requirements:

  • Windows 10 or 11 operating system
  • 4GB RAM (8GB recommended)
  • 2GB free disk space
  • Intel Core i3 or equivalent (Core i5 or better recommended)
  • Internet connection

Download and Installation

Follow these steps to install KodKit:

  1. Download the KodKit installer from our download page.
  2. Once the download is complete, locate the installer file in your downloads folder.
  3. Double-click the installer file to begin the installation process.
  4. If prompted by Windows security, click "Yes" to allow the installer to make changes to your device.
  5. In the installer window, click "Next" to continue.
  6. Read and accept the license agreement, then click "Next".
  7. Choose the installation location or use the default, then click "Next".
  8. Select any additional components you want to install, then click "Next".
  9. Click "Install" to begin the installation process.
  10. Wait for the installation to complete.
  11. Click "Finish" to complete the installation.

Note: If you encounter any issues during installation, try running the installer as an administrator by right-clicking the installer file and selecting "Run as administrator".

First Launch

After installation, you can launch KodKit by:

  • Double-clicking the KodKit icon on your desktop
  • Finding KodKit in your Start menu
  • Using the Windows search function to find "KodKit"

When you first launch KodKit, you may be prompted to:

  • Complete a brief tutorial (recommended for new users)

Tip: Kodkit will be upgraded by time, just let us know by contacting to us..

Updating KodKit

KodKit will automatically check for updates when you start the application (if you're connected to the internet). You can also manually check for updates by:

  1. Opening KodKit
  2. Clicking on "Help" in the top menu
  3. Selecting "Check for Updates"

Uninstalling KodKit

If you need to uninstall KodKit, you can do so through the Windows Control Panel:

  1. Open the Control Panel
  2. Select "Programs and Features" or "Apps & features" (depending on your Windows version)
  3. Find KodKit in the list of installed programs
  4. Select KodKit and click "Uninstall"
  5. Follow the prompts to complete the uninstallation

Alternatively, you can use the uninstaller included with KodKit:

  1. Open the Start menu
  2. Find the KodKit folder
  3. Click on "Uninstall KodKit"
  4. Follow the prompts to complete the uninstallation

Note: Uninstalling KodKit will not delete your saved projects by default. Your projects are stored in your Documents folder under wherever you save them.

Need Help?

If you encounter any issues during installation or have questions, please visit our FAQ page or contact our support team.
